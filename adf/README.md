# ADF Pipeline Definitions

This directory contains Azure Data Factory pipeline JSON exports.

Place your ADF pipeline JSON files here, organized by pipeline name:

```
adf/
├── PL_Bronze_Event_Master.json
├── PL_Bronze_To_Standardized_Parquet.json
├── PL_Oeil_Guardian.json
├── PL_Oeil_Core.json
├── PL_Oeil_Quality_Engine.json
└── ...
```

These files serve as version-controlled documentation of your ADF orchestration layer.
